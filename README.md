# MixTools (MixToolBox)

**作者**: MuyouHCD
**版本**: 4.8.69
**Blender 版本**: 3.6+
**分类**: Object

MixTools 是一个 Blender 综合工具箱插件，集成了大量批量操作、场景管理、材质处理、动画工具等功能，旨在提升 Blender 的使用效率和工作流程。插件面板位于 3D 视口侧栏的「工具箱」标签页，顶部提供搜索框，可快速定位任意功能。

---

## 安装与依赖

### 安装方法
1. 下载插件文件夹
2. 在 Blender 中打开 **编辑 → 偏好设置 → 插件**
3. 点击 **安装** 并选择插件压缩包
4. 启用 **MixToolBox** 插件

### 依赖库
- **Pillow (PIL)** — 用于渲染后处理、图像边框添加等功能
  - 插件激活时会自动从 `package/` 目录安装本地依赖包
  - 若自动安装失败，可手动安装：
    ```
    cd [Blender安装目录]/[版本号]/python/bin
    python -m pip install pillow
    ```
  - 缺少 Pillow 时插件仍可运行（受限模式），仅渲染后处理等图像功能不可用

### 在线更新
插件支持在线更新功能（通过偏好设置面板）。更新会覆盖所有文件，**请提前备份 `RigJson/` 中的自定义配置文件**。
